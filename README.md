&#x20;PÁGINA WEB PERSONAL (HTML, CSS, JAVASCRIPT)



&#x20;Ver sitio: https://lean-iv.github.io/



Proyecto personal desarrollado con HTML, CSS y JavaScript que funciona como sitio de presentación profesional y portfolio web.
La página centraliza información clave del perfil, incluyendo CV, conocimientos técnicos, proyectos y medios de contacto, con el objetivo de ofrecer una visión clara y accesible para reclutadores y entornos laborales.



&#x20;OBJETIVOS DEL PROYECTO



Simular un sitio web personal que permita:

* Presentar información profesional de forma clara y ordenada
* Centralizar el acceso al CV y medios de contacto
* Mostrar conocimientos técnicos de manera visual e interactiva
* Reunir proyectos y enlaces relevantes en un solo espacio
* Contar con una presencia online propia utilizando GitHub Pages
* Aplicar una estructura simple, mantenible y fácil de navegar



El foco del proyecto está puesto en la presentación del perfil, la organización del contenido y la construcción de una web estática funcional.



&#x20;CONCEPTOS APLICADOS



* Estructuración de páginas con HTML
* Diseño y estilos visuales con CSS
* Interactividad con JavaScript
* Manipulación del DOM
* Renderizado dinámico de contenido
* Integración de PDFs en la web
* Organización de recursos estáticos
* Navegación simple orientada a portfolio
* Publicación web mediante GitHub Pages



&#x20;ESTRUCTURA DEL PROYECTO



* index.html: contiene la estructura principal del sitio
* css/: incluye los estilos visuales (styles.css)
* js/: contiene la lógica e interactividad del sitio (script.js)
* assets/: almacena recursos como imágenes y archivos PDF utilizados por la web
* Herramientas/: carpeta auxiliar con recursos relacionados al desarrollo



&#x20;FUNCIONALIDADES DESTACADAS



* Presentación personal y profesional en una sola página
* Acceso directo al CV
* Sección de conocimientos técnicos
* Carga dinámica de PDFs para mostrar material de conocimientos
* Enlace a proyectos de GitHub
* Acceso rápido a contacto por WhatsApp, LinkedIn y email
* Sitio publicado online mediante GitHub Pages
